# Cagit ApS Website

This repository contains the static website for Cagit ApS.

The site is built with plain HTML and CSS. It has no backend, no build step, no
JavaScript dependencies, no analytics and no tracking.

## Hosting

The site can be hosted directly with GitHub Pages from the root of the `main`
branch.

The configured custom domain is:

```text
cagit.dk
```

## Files

- `index.html` - company homepage
- `privacy.html` - privacy policy
- `support.html` - support information
- `terms.html` - terms of service
- `styles.css` - shared styling
- `CNAME` - GitHub Pages custom domain
- `.nojekyll` - disables Jekyll processing on GitHub Pages
